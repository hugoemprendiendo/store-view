# **App Name**: StoreView

## Core Features:

- Branch Management: Module to manage and list all the chain's branches.
- Incident Reporting: Module to report incidents with photo, audio (transcribed to text), text, priority and category. Includes an 'Analyze Incident' tool that uses AI to suggest a title, category, priority , and status (defaults to 'open').
- Dashboard: Dashboard displaying all branches represented by an image that changes color according to the status of the branch incidents, filterable by brand and region.
- Branch Detail: Clicking on a branch on the dashboard shows the branch details and a 'Report Incident' button.
- Incident Detail: Module to view incident details and change the incident status.
- Configuration: Area to manage incident categories, status and priority in a list for selection in a dropdown list during incident reporting.

## Style Guidelines:

- Primary color: Dark blue (#2c3e50) to convey a sense of trust and reliability.
- Background color: Light gray (#f0f2f5) for a clean and modern look.
- Accent color: Orange (#e67e22) for 'Report Incident' buttons and highlighting important information.
- Body and headline font: 'Inter', a sans-serif font for a modern and clean user interface.
- Use clear and concise icons to represent branch status and incident categories.
- Dashboard layout featuring a grid of branch images that change color according to incident status.
- Subtle transitions when the status of a branch changes on the dashboard.
